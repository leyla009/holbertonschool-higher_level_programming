==================================
How to use matrix_divided function
==================================

Import the function:
>>> matrix_divided = __import__('2-matrix_divided').matrix_divided

Normal case:
>>> matrix = [[1, 2, 3], [4, 5, 6]]
>>> matrix_divided(matrix, 3)
[[0.33, 0.67, 1.0], [1.33, 1.67, 2.0]]

Check if the original matrix is unchanged:
>>> matrix
[[1, 2, 3], [4, 5, 6]]

Divisor is a float:
>>> matrix_divided([[10, 20]], 2.5)
[[4.0, 8.0]]

Each row must be the same size:
>>> matrix_divided([[1, 2], [3]], 2)
