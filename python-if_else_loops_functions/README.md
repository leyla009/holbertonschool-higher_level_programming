# Python - if/else, loops, functions

## Description
This project is part of the Higher Level Programming curriculum at Holberton School. It covers the basics of Python programming, including conditional statements (`if`, `if...else`), loops (`while`, `for`), and an introduction to functions.

## Learning Objectives
* Why indentation is so important in Python
* How to use the `if`, `if ... else` statements
* How to use comments
* How to assign values to variables
* How to use the `while` and `for` loops
* How to use the `break` and `continue` statements
* How to use `else` clauses on loops
* What the `pass` statement does and when to use it
* How to use `range`
* What is a function and how to use functions
* What a function returns when it does not use a `return` statement
* Scope of variables
* What is a traceback
* What are the arithmetic operators and how to use them

## Requirements
* Allowed editors: `vi`, `vim`, `emacs`
* All files interpreted/compiled on Ubuntu 20.04 LTS using `python3` (version 3.8.5)
* All files should end with a new line
* The first line of all files should be exactly `#!/usr/bin/python3`
* Code should use the `pycodestyle` (version 2.8)
