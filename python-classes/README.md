# Python - Classes and Objects

## Project Description
This directory contains projects completed as part of the higher-level programming curriculum. It focuses on the fundamentals of **Object-Oriented Programming (OOP)** in Python. The goal is to understand how to bundle data and functionality together, creating custom types that model real-world concepts.

---

## Learning Objectives
By completing the projects in this directory, the following concepts are covered:
* Why Python programming is awesome
* The difference between Object-Oriented Programming (OOP) and procedural programming
* What is a class and what is an object (instance)
* The difference between a class and an object/instance
* What are attributes (both instance and class attributes)
* What is the `__init__` method and how to use it
* What is data abstraction, data encapsulation, and information hiding
* What is a property and the difference between an attribute and a property in Python
* The Pythonic way to use getters and setters
* What is the `__str__` method and how to use it
* How to dynamically create arbitrary new attributes for existing instances

---

## Requirements
* All files are interpreted/compiled on Ubuntu 20.04 LTS using `python3` (version 3.8.5)
* All files should end with a new line
* The first line of all files should be exactly `#!/usr/bin/python3`
* Code should use the `pycodestyle` style (version 2.8.*)
* All modules, classes, and functions must have documentation (`__doc__`)

---

## Directory Structure

| File | Description |
| :--- | :--- |
| `0-square.py` | Defines an empty class `Square`. |
| `1-square.py` | Defines a class `Square` with a private instance attribute `size`. |
| `2-square.py` | Adds size validation to the `Square` class. |
| `3-square.py` | Adds a public instance method to calculate the area of the square. |
| `4-square.py` | Implements getters and setters for the `size` attribute. |
| `5-square.py` | Adds a method to print the square to stdout using the `#` character. |
| `6-square.py` | Adds a `position` attribute and handles coordinate-based printing. |
