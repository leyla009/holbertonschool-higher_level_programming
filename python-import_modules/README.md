# Python - import & modules

## Description
This project focuses on understanding how to organize Python code using modules and packages. It covers importing functions from other files, handling command-line arguments using the `sys` module, and using the `__name__` variable to prevent code execution during imports.

## Learning Objectives
* How to import functions from other files
* How to use the `import` and `from ... import` statements
* How to use `as` to alias imported names
* How to use `dir()` to list the contents of a module
* How to prevent code from executing when imported using `if __name__ == "__main__":`
* How to use command-line arguments with `sys.argv`

## Files

| File | Description |
| :--- | :--- |
| `0-add.py` | Imports an addition function and prints the result. |
| `1-calculation.py` | Imports math functions (add, sub, mul, div) from another file. |
| `2-args.py` | Prints the number of and the list of its arguments. |
| `3-infinite_add.py` | Prints the result of the addition of all arguments. |
| `4-hidden_discovery.py` | Prints all names defined by a compiled module (hidden_4.pyc). |
| `5-variable_load.py` | Imports a variable from a file and prints its value. |

## Requirements
* Allowed editors: `vi`, `vim`, `emacs`
* All files interpreted/compiled on Ubuntu 20.04 LTS using `python3` (version 3.8.5)
* All files must end with a new line
* The first line of all files should be exactly `#!/usr/bin/python3`
* Code should use the `pycodestyle` (version 2.8.*)
* All files must be executable
