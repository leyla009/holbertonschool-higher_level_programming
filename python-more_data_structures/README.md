# Python - More Data Structures: Set, Dictionary

## Description
This project focuses on Sets, Dictionaries, and functional programming tools like Lambda, Map, and Filter in Python.

## Requirements
- **OS:** Ubuntu 20.04 LTS
- **Language:** Python 3.8.5
- **Style:** pycodestyle 2.8.*

## Task Overview
| File | Task | Description |
|---|---|---|
| 0-square_matrix_simple.py | Square Matrix | Computes the square of all integers in a matrix. |
| 1-search_replace.py | Search & Replace | Replaces all occurrences of an element in a new list. |
| 2-uniq_add.py | Unique Addition | Adds all unique integers in a list. |
| 3-common_elements.py | Common | Returns a set of common elements in two sets. |
| 4-only_diff_elements.py | Difference | Returns a set of elements present in only one set. |
| 5-number_keys.py | Key Count | Returns the number of keys in a dictionary. |
| 6-print_sorted_dict.py | Sorted Print | Prints a dictionary by ordered keys. |
| 7-update_dictionary.py | Update | Replaces or adds key/value in a dictionary. |
| 8-simple_delete.py | Delete | Deletes a key in a dictionary. |
| 9-multiply_by_2.py | Multiply | Returns a new dictionary with values multiplied by 2. |
| 10-best_score.py | Best Score | Returns the key with the highest integer value. |
| 11-multiply_list_map.py | Map Multiplies List Values using map |
| 12-roman_to_int.py | Roman to Int Conversion | Converts a Roman numeral string to an integer.

## Key Concepts
- **Sets:** Unordered collections of unique elements.
- **Dictionaries:** Key-value pairs for efficient data retrieval.
- **Lambda:** Anonymous, one-line functions.
- **Map/Filter:** Functional tools for iterating and transforming data.
